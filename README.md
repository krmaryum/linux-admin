# Linux Administration

A complete Linux Administration learning repository with study notes, hands-on labs, RHCSA preparation, Bash scripting, troubleshooting guides, interactive MCQs, and real-world scenarios.

---

## Repository Purpose

This repository is designed to build strong Linux Administration skills from beginner to advanced level.

It includes:

- Linux fundamentals
- User and group management
- Permissions and ACLs
- SELinux
- File descriptors and redirection
- Shell and environment concepts
- Bash scripting
- Networking
- Systemd
- Storage and LVM
- RHCSA-style labs
- Interactive HTML quizzes
- Real-world troubleshooting scenarios
